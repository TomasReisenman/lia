🤖 Root Agent

root_agent

gemini-3.1-flash-lite · Orquestador principal

Delega al sub-agente especializado según la intención detectada.

🗣️ "explícame redes neuronales" → Wikipedia
🗣️ "busca proyectos transformers en GitHub" → GitHub
🗣️ "hazme un ejercicio de ML" → Ejercicios
🗣️ "mi respuesta es B" → Corrector
🗣️ "qué debería estudiar?" → Study Advisor
🗣️ "https://..." → URL Reader
root_agent = Agent(
    model='gemini-3.1-flash-lite',
    name='root_agent',
    sub_agents=[wiki, github, exercises,
                corrector, study_advisor, url_reader],
)

📖 Wikipedia Agent

wikipedia_agent

MCP · wikipedia-mcp --language es

Investiga conceptos académicos usando la API de Wikipedia en español. Nunca responde con conocimiento propio — siempre usa las herramientas MCP.

Herramientas MCP

  • search_wikipedia
  • get_article
  • get_summary
  • get_sections
  • get_related_topics
  • extract_key_facts

Flujo

  1. Buscar el concepto
  2. Refinar si es necesario
  3. Leer artículo completo
  4. Resumir en español
McpToolset(
    connection_params=StdioConnectionParams(
        server_params=StdioServerParameters(
            command="wikipedia-mcp",
            args=["--language", "es"],
        ),
    ),
)

🐙 GitHub Agent

github_agent

MCP · StreamableHTTP · GH_TOKEN

Busca repositorios, proyectos y código en GitHub usando el MCP de Copilot. Autenticado via GH_TOKEN.

connection_params = StreamableHttpConnectionParams(
    url="https://api.githubcopilot.com/mcp/",
    headers={"Authorization": f"Bearer {os.getenv('GH_TOKEN')}"},
)
Respuesta típica: nombre del repo, descripción, ⭐ estrellas, URL

✅ Corrector Agent

corrector_agent

Session State · Comparación de respuestas

Lee el ejercicio guardado por exercises_agent, compara la respuesta y da retroalimentación.

def get_exercise_data(tool_context):
    return {
        "title": tool_context.state.get("exercise_title"),
        "question": tool_context.state.get("exercise_question"),
        "options": tool_context.state.get("exercise_options"),
        "correct_answer": tool_context.state.get("exercise_correct_answer"),
        "explanation": tool_context.state.get("exercise_explanation"),
    }

Flujo: respuesta usuario → get_exercise_data() → compara → feedback

def save_summary(url, title, summary, tool_context):
    tool_context.state["last_url"] = url
    tool_context.state["last_url_title"] = title
    tool_context.state["last_url_summary"] = summary

Procesamiento de HTML

class _HTMLStripper(HTMLParser):
    def handle_starttag(self, tag, attrs):
        if tag in ("script", "style"):
            self._skip = True
    def handle_data(self, data):
        if not self._skip:
            self._text.append(data)
    def get_text(self):
        return re.sub(r'\s+', ' ', ' '.join(self._text)).strip()

Excluye <script> y <style>, normaliza espacios, trunca a 8000 caracteres.

🚀 Cómo ejecutar

# 1. Instalar dependencias
pip install -r requirements.txt

# 2. Configurar variables de entorno (.env)
GH_TOKEN=ghp_...
HF_TOKEN=hf_...

# 3. Inicializar FAISS (si no existe)
python scripts/embed.py info/

# 4. Ejecutar la aplicación adk
adk run lia

Los agentes se activan automáticamente según la intención detectada en cada mensaje del usuario.
